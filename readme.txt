This Plugin provides you functionality so you can submit ratings for employees. which can further be used to evaluate employee performance. Ratings been done on 5 pre-defined Parameters.
 
== Description ==
 
= This Plugin is a simple employee ratings and time sheet system. It lets you rate employees based on 5 pre-defined parameters which are 

1. Professionalism
2. Efficiency and technical knowledge
3. Helping your Teammates.
4. Proactiveness
5. Leaves

By Timesheet your employees can keep track of time. The amount of time they spent on any project or on any task in a day.

In Reports you can check how your employees performing based on these ratings. These reports are based on month and Days basis and get you Current average Rating of employee.

In Timesheet Reports you can check timesheet report of any employee on any date.

As per next Phase we going to make Those 5 parameters to be dynamic which organisation will be able to control based on their own Ratings system keys.

== Installation ==
 
1. Upload `navthemes-employee-ratings` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
